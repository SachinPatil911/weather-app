# Weather App

A beautiful, modern weather application using OpenWeatherMap API.

## Features
- Search weather by city name
- Real-time temperature display
- Humidity, wind speed, and "feels like" temperature
- Dynamic weather icons
- Beautiful glass-morphism UI design
- Save API key locally (stored in browser)

## How to Get API Key

1. Visit [OpenWeatherMap API](https://openweathermap.org/api)
2. Sign up for a free account
3. Navigate to "API Keys" section
4. Copy your API key

## How to Use

1. Open `index.html` in your browser
2. Enter your OpenWeatherMap API key in the input field
3. Click "Save API Key" button
4. Enter a city name in the search box
5. Click search or press Enter

## Files
- `index.html` - Main HTML structure
- `style.css` - Styling and animations
- `script.js` - JavaScript logic and API integration

## Technologies
- HTML5
- CSS3 (Glass-morphism, Animations)
- JavaScript (Fetch API, LocalStorage)
- OpenWeatherMap API
